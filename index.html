코딩 마스터의 특징

코딩 마스터는 다른 사이트와는 차원이 다른 깔끔한 구상을 보여줍니다. 모든 페이지가 사용자가 접속하고 읽기 쉽도록 정교하게 구성되어 있습니다.

폭넓은 강사진

코딩 마스터는 정말로 뛰어난 개발자만을 강사진으로 뽑습니다. 각각의 강사진은 현직 개발자로서 프로그래밍을 접하는 학생에게 친절하게 알려줍니다.

핵심적인 강의

강의는 실제 개발에 반드시 핵심적인 내용을 중심으로 합니다. 강의를 듣는 시간 내내 요약된 정보를 온전히 받아들일 수 있습니다.

  최신 강의 목록

C언어 기초 프로그래밍 강의 New

C언어 강의는 기초 프로그래밍 강의입니다. 처음 프로그램을 접하는 입문자가 듣기에 적합한 강의 입니다. 강의는 무료이며 C언어 기초 프로그래밍 강의는 총 20강으로 구성 됩니다.

JAVA 기초 프로그래밍 강의 New

JAVA 강의는 '객체' 지향 프로그래밍의 기본을 알려주는 강의입니다. 처음 프로그래밍을 접하는 입문자가 듣기에 적합한 강의입니다. 강의는 무료이며 JAVA 기초 프로그래밍 강좌는 총 30강으로 구성됩니다.

안드로이드 스튜디오 활용 강의 New

안드로이드 강의는 JAVA를 활용하여 안드로이드 애플리케이션을 개발하는 강의입니다. 속도감 있는 강으로 앱 개발 입문자가 빠르게 앱을 만들어 보면서 기본 개념을 단단히 다지도록 합니다.
